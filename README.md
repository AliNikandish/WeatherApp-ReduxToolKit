## Installation

#### Clone the project

```bash
  git clone https://github.com/AliNikandish/WeatherApp-ReduxToolKit.git
```

#### Go to the project directory

```bash
  cd WeatherApp-ReduxToolKit
```

#### Install QuizApp with npm

```bash
  npm install
```

#### Change API key

```bash
  Go To src Then apiKey.ts And Paste Your Api Key
```
#### Run:
```bash
  npm run dev
```
